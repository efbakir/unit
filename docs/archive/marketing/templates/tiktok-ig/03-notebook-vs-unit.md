# TikTok/IG template — Notebook vs Unit

> **Cadence**: 1 every 2 weeks. **Don't burn this out** — it's Unit's defining claim.
> **Length**: 20-30s vertical.
> **Source**: Source A (DIY) — needs you doing the comparison live.

---

## Structure

| Time | Action |
|---|---|
| **0-3s** | Hand writing in notebook with stopwatch visible |
| **3-15s** | Cut to Unit — ghost-value tap, haptic, done in 2.4s on overlay timer |
| **15-25s** | Side-by-side numbers — notebook 8.1s, Unit 2.4s |
| **25-30s** | Close — *"Faster than paper. Free on App Store."* |

The visible timer + the speed reveal is the entire content. Specificity > pretty.

---

## Hook variants (first 3s)

**A** (timer-first):
> [Stopwatch counting up. Hand writing in notebook. Pen scratching.]

**B** (text overlay-first):
> [Text: "I tracked 200 sets in this notebook because every gym app was slower."]

**C** (deadpan voiceover):
> *"Notebook took me 8 seconds. App takes me 2."*

---

## Recording setup

**Notebook portion** (~3-8s):
- Real gym notebook, real pen
- Real recent set in your handwriting (last week's lift)
- Stopwatch / timer running on screen above OR a clip-on stopwatch app
- One steady camera angle

**Unit portion** (~7-12s):
- iPhone screen-record while you log a real set in Unit
- Same exercise as the notebook entry (so the comparison is fair)
- iOS Control Center screen recording → import to CapCut
- Add a timer overlay synced to actual log time

**Side-by-side** (~10s):
- CapCut split-screen: notebook on left, Unit on right
- OR sequential: notebook full screen → cut to Unit full screen
- Show the two times: "8.1s" vs "2.4s"

---

## Voiceover (optional — works without one too)

Keep it dry, calm, ~10-15 words:

```
Notebook: 8.1 seconds.
Unit: 2.4 seconds.
Three taps. Ghost value pre-fills. Done.
Free on App Store.
```

If you skip voiceover, let the timer + side-by-side carry it. Silence works for this format.

---

## Closing

- *"Faster than paper. Free on App Store."* (text overlay or voiceover)
- OR silence + app icon close

---

## Why this format works

Unit's defining claim from `docs/launch-plan.md` §1: *"faster than paper, smarter than Notes, your gym notebook upgraded."*

The notebook-vs-app comparison is the strongest visual proof of that claim. Visible timers + real handwriting + real screen-record = unfakeable.

Per `research/viral-patterns-2026-04-29.md` §2 hooks-that-work: "A timer counting up while you log a real set" is one of the consistently-working first-3s patterns.

## Why we cap at 1/2 weeks

If you post this every week, the format becomes a meme — it stops feeling like proof and starts feeling like a brand catchphrase. Once every 2 weeks keeps it fresh.

## Variants over time

Rotate the notebook entry between exercises so each post has a different lift:
- Week 1: bench press, 5×5
- Week 3: squat, 5×3
- Week 5: deadlift, 3×5
- Week 7: overhead press, 5×5
- Week 9: row, 5×8

Audience that watches multiple posts sees the comparison hold across exercises.

## Recording checklist

- [ ] Notebook + pen ready, real recent set logged
- [ ] Stopwatch / timer visible
- [ ] iPhone screen-record set up for Unit portion
- [ ] Tripod for steady camera
- [ ] Same exercise in both portions

## Post-production

- [ ] CapCut: split-screen or sequential
- [ ] Timer overlay synced to actual log time (not faked)
- [ ] Submagic captions if voiceover present
- [ ] Closing screen with app icon
- [ ] 1080×1920 vertical

---

## Don't do

- ❌ Fake the timer numbers — audience can tell
- ❌ Cherry-pick the notebook entry to make it look slower than it really is
- ❌ Use a different exercise on each side (unfair comparison)
- ❌ Add music with lyrics
- ❌ Speed up the notebook side (slop tell)

The whole format depends on the timer being real. If you fake it once and someone calls it out, the format dies.
