# TikTok/IG template — Tradesman demo

> **Cadence**: 1/week. Cornerstone format.
> **Length**: 30-60s vertical (9:16, 1080×1920).
> **Source**: Source A (DIY) or Source B (UGC) per `content-engine.md`. Source C (ElevenLabs) optional.

---

## Structure

| Time | Action | Notes |
|---|---|---|
| **0-3s** | Screen-record of a real set being logged with timer overlay | Hook MUST land in first 3s. No intro, no "hi guys". |
| **3-25s** | Voiceover narrating: ghost value, tap to confirm, haptic, done | Calm, dry, no music. ~30 words max. |
| **25-50s** | One specific differentiator vs competitors | "Hevy makes you tap through 4 fields. Strong needs 3. This is one." |
| **50-60s** | Closing screen with app icon + name | Simple text overlay: app name + "Free on App Store" |

---

## Hook variants (first 3 seconds — pick one per video)

**A** (timer-led):
> [Timer overlay counting up: 1.2s… 1.8s… set logged]
> *No voice. Let the visual carry the hook.*

**B** (split-screen):
> [Left: bloated competitor onboarding screen. Right: Unit's 3-step start.]
> *Voiceover: "10 screens vs 1."*

**C** (frustration-led):
> [Cold open: hands sweaty, you fail to tap a tiny number stepper in [Hevy/Strong/Liftosaur]. Frustration.]
> *Voiceover: "Logging a set under fatigue. Watch this."*

**D** (text-led):
> [Text overlay over screen demo: "Logging a set in 2.4 seconds. One-handed. Sweaty."]

---

## Voiceover script (30-60s, dry, calm)

```
Three taps. Ghost value from last session pre-fills weight and reps.
Tap to confirm. Haptic. Done.

That's the whole loop. No menus. No AI coach. No "what should you
lift today." Just the set you actually did.

I built this because every gym app made me tap through 4 fields under
fatigue. This is one tap. The previous session pre-fills. You
confirm or you adjust. Then you log.

[Free on App Store. Link in bio.]
```

Trim to fit length. Cut ruthlessly — script over 60s = too long.

---

## CTA

End with one of:
- *"Free on App Store. Link in bio."*
- *"Link in bio."*
- Silence, close on app icon

❌ Never:
- *"Smash the download button"*
- *"Comment for the link"*
- Voice CTAs (kill retention)

## Why this format works

Per `research/viral-patterns-2026-04-29.md` §2a:
> The screen recording + calm voiceover format gets disproportionate watch-time on TikTok because the algorithm weighs completion rate ≥70% in first 3s. A 12-second screen-record of logging a set with ghost values, voiceover "no menus, no AI coach, three taps under fatigue" — that's the format. Realistic conversion: 0.3–1% of views to TestFlight signup if there's a pinned link in bio.

The "tradesman" framing matches `PRODUCT.md` brand voice (calm, expert, honest) and the existing `docs/launch-plan.md` §4 rule: *"every video shows the app running in a real gym on a real iPhone."*

## Don't do

- ❌ Music with lyrics (distracts from screen demo)
- ❌ Voice intro ("Hey guys, today I'm going to show you…") — kills 3s retention
- ❌ "Day 1238 of building" framing
- ❌ Generic motivational quotes
- ❌ AI talking head as the demo subject
